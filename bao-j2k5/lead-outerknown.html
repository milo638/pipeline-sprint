Outerknown

Sustainable surf label juggling $400 Journey outerwear, Outerworn resale, and wholesale spikes without matching PDPs or lifecycle.

Revenue est. $30M Shopify Plus Priority: High

Why This Matters Now

Outerknown is flooding channels with $400 Journey outerwear, yet the PDP is still a sustainability blog post. Shoppers never see performance proof or bundle logic, so they bounce before even scrolling to the add-to-cart.

Killer Insight

The Journey hero has never been tested against a spec-first variant; it still opens with ethos copy, so shoppers never see measurable proof before the CTA.

What’s happening

  • Hero above the fold is 100% ethos copy; no wind/rain stats, no lifestyle images showing fit.
  • Mobile gallery uses 4MB lifestyle images with zero zoom controls or fabric callouts.
  • Cross-sell blocks promote random tees instead of layering pieces / accessories that increase AOV.

What to fix

  • Redesign the hero with performance badges, temp-range callouts, and an above-fold CTA so specs land before any sustainability copy.
  • Drop a segmented spec + fit module (icons + slider) directly under the buy box, replacing the buried accordion.
  • Swap the generic cross-sell for a Journey bundle grid (jacket + layer + bag) with quick-add buttons to raise AOV.
